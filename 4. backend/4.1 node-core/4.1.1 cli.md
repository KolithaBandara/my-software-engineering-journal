# CLI

## Demystifying the Command Line

### 1. pwd
* This command shows us our current working directory.

### 2. ls
* Using this you can list the items on your working directory.

### 3. cd
* You can navigate with this command.

### 4. clear
* Using this command, you can clear the command line interface.

## Create & Delete files

* create - `touch cat.txt`
* delete - `rm cat.txt` - rm means remove

## Create & Delete directories

* Create - `mkdir animals`
* Delete - `rmdir animals`/ `rm -r animals`

## Write to files

* `echo "Hello World!` - prints "Hello World!" inside of the terminal.

## Read from files
* command - `cat hello.txt`
prints "hello world!" to the terminal.
